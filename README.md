🇧🇷

Resolve Organizer é uma ferramenta de suporte ao workflow para editores e coloristas que utilizam o DaVinci Resolve. O objetivo principal é eliminar o trabalho manual repetitivo na organização de pastas de mídia, automatizando a estrutura de Bins para novos projetos.

Funcionalidades Principais

Estruturação de Projetos: Cria rapidamente hierarquias padronizadas de pastas (Bins), como Footage, Audio, Graphics, Proxies e Timelines.
Padronização de Workflow: Garante que todos os projetos sigam a mesma lógica de organização, facilitando a colaboração e a entrega de arquivos.
Interface Direta: Webapp leve focado em gerar a estrutura necessária para importação ou integração com o software da Blackmagic Design.
Ganho de Produtividade: Reduz o tempo de configuração inicial do projeto, permitindo que o editor comece a trabalhar imediatamente.

🇬🇧 English

Resolve Organizer is a workflow support tool for editors and colorists using DaVinci Resolve. Its main goal is to eliminate repetitive manual work when organizing media folders by automating the Bin structure for new projects.

Key Features

Project Structuring: Quickly creates standardized folder (Bin) hierarchies (e.g., Footage, Audio, Graphics, Proxies, Timelines).
Workflow Standardization: Ensures all projects follow the same organizational logic, making collaboration and file delivery easier.
Direct Interface: A lightweight web app focused on generating the required structure for import or integration with Blackmagic Design software.
Productivity Boost: Reduces initial project setup time, allowing the editor to start editing immediately.
